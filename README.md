# Implementation of John Conway's Game of Life

### Building
```bash
cd conways-game-of-life && cmake -DDISPLAY_TYPE=curses -DCMAKE_BUILD_TYPE=Release -Bbuild/ -S./ && cmake --build build/ -j8
```
